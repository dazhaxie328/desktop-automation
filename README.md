# 智能桌面自动化机器人

通过 Telegram 自然语言控制电脑，支持复杂任务自动执行。

## 功能特性

- ✅ 自然语言命令理解
- ✅ 鼠标操作（点击、移动、拖拽）
- ✅ 键盘操作（输入、快捷键）
- ✅ 屏幕截图
- ✅ 文件/应用打开
- ✅ 终端命令执行
- ✅ 循环执行
- ✅ 定时执行
- ✅ 任务历史记录

## 快速开始

### 1. 配置 Telegram

在 `~/crypto-monitor/.env` 文件中：

```
TELEGRAM_BOT_TOKEN=你的Bot Token
TELEGRAM_CHAT_ID=你的Chat ID
```

### 2. 启动机器人

```bash
cd ~/desktop-automation
chmod +x start_smart_bot.sh
./start_smart_bot.sh
```

## 安全设置

- 鼠标移到屏幕左上角触发安全异常
- 危险命令被拒绝执行
- 命令执行超时限制 (30秒)
